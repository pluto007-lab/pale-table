# pale-table

![トップページ](img/preview.png)

「余白のある暮らしに、淡い器を。」

架空の食器ブランド「pale table」のランディングページ（LP）です。
白・水色・淡いベージュを基調としたやさしい世界観を意識し、商品の魅力が伝わるシンプルなデザインを制作しました。

## 制作目的

- ランディングページ（LP）の制作練習
- HTML・CSS・JavaScriptを用いたWebサイト制作
- GitHub Pagesでの公開

## ターゲット

- ナチュラルな暮らしが好きな方
- シンプルでやさしいデザインが好きな方
- 食器やインテリアにこだわりのある方

## 使用技術

- HTML5
- CSS3
- JavaScript
- GitHub Pages

## 工夫したポイント

- 白・水色・淡いベージュを基調とした統一感のあるデザイン
- 商品写真が引き立つシンプルなレイアウト
- ファーストビューでブランドイメージが伝わる構成
- レスポンシブ対応を意識したレイアウト
